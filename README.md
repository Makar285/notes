# Пользовать входит
### 1) Пользовать заходит на сайт
### 2) Смотрится localStorage и загружаются данные от туда
### 3) Добавляются все нужные обработчики событий

# Пользовать нажимает добавить заметку
### 1) Получение данных из формы и сохранение их
### 2) Очищение значений тегов input
### 3) Если данные валидные то создать разметку и добавить ее на страницу в раздел неопределённой важности
### 4) Если данные не валидные вывести сообщение об ошибке вк

# Пользователь нажимает удалить заметку
### 1) Получить данные из localStorage
### 2) Изменить в нужной заметке значение isDeleted с false по умолчанию, на true
### 3) Обновить данные в localStorage
### 4) Отрендерить заметки 

# Пользователь нажимает изменить заметку
### 1) Получить данные из localStorage
### 2) Изменить их
### 3) Обновить данные в localStorage
### 4) Отрендерить заметки 

# Пользователь нажимает изменить заметку
### 1) Сохранить данные который находятся в title и content
### 2) Удалить заголок и абзац
### 3) Добавить на из месте две формы с .value = title и content 
### 4) Добавить кнопку сохранить, при нажатии на которую данные из создных двух форм будут сохраняться, две формы с кнопкой будут удаляться и на их месте будут появляться h2 и p со значениями сохранными в переменную
### 5) Сохранить изменения в localStorage, отрендерить новые данные
### 6) Если пользователь нажмет кнопку отмены изменения заметки то вернуть состояние замтеку в то в котором она была до нажатия на кнопку изменения заметки


# Пользователь нажимает удалить заметку
### 1) Получить элемент на который нажали, e.target(элемент на которой нажали, перетащили и тд)
### 2) Получить из кнопки данные data-note-id по айди заметки
### 3) изменить его isDeleted на true


# При загрузке странице
### 1) Получить данные из localStorage
### 2) Если они есть то в функции в model которая отвечает за эту функциональность вернуть эти данные и отобразить их на странице
### 3) Если их нету то в функции в model которая отвечает за эту функциональность вернуть undefined
### 4) В controller вызвать эту функцию и проверить результат если он есть то отрендерить новые данные, а если нету то ничего не делать
